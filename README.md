# Projet ETL Batch – Analyse des ventes

##  Objectif
Construire un pipeline ETL batch pour analyser les ventes
et fournir un dashboard BI interactif.

## Stack technique
- PostgreSQL
- SQL
- Python
- Docker
- Metabase

##  Pipeline ETL
1. Extraction des données brutes
2. Nettoyage et transformation
3. Chargement dans PostgreSQL
4. Visualisation avec Metabase

##  Indicateurs clés
- Top 5 produits les plus vendus
- Chiffre d’affaires par pays
- Évolution mensuelle des ventes
- Filtres dynamiques par période
